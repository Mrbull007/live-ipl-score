# 🏏 Live IPL Score Tracker

A lightweight Python script that fetches and displays **live IPL scores** in your terminal with auto-refresh every 15 seconds.

## Features
- Shows only IPL matches
- Auto-refreshes every 15 seconds
- Clean terminal output with timestamps
- Uses web scraping from Cricbuzz (no API key needed)

## Requirements
- Python 3.8+
- `requests`
- `beautifulsoup4`

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/live-ipl-score.git
cd live-ipl-score
pip install -r requirements.txt
